# macOS 快捷键

<!-- more -->

从 Windows 过渡到 macOS

-   Win == Command `<C>`
-   Alt == Option `<O>`
-   Shift == Shift `<S>`
-   Delete == Backspace `<B>`

1. 复制 `<C-c>`
2. 粘贴 `<C-v>`
3. 复制+剪切 `<C-O-v>`
4. 撤销 `<C-z>`
5. 重做 `<C-S-z>`
6. 删除 `<C-B>`
7. 彻底删除 `<C-S-B>`
