## Premiere Pro

官方文档: [Keyboard shortcuts in Premiere Pro](https://helpx.adobe.com/premiere-pro/using/keyboard-shortcuts.html)

### 编辑快捷键

1. `Ctrl + Z`: Undo
2. `Ctrl + Shift + Z`: Redo

### 剪辑快捷键

1. `C`: Razor Tool
2. `V`: Selectetion Tool
3. `M`: Add Marker
4. `I`: Mark In
5. `O`: Mark Out
6. `Shift + I`: Go to In
7. `Shift + O`: Go to Out
8. `Ctrl + K`: Add Edit 从当前位置剪开
